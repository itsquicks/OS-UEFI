Built-In Commands
___________________________________________________________________________________________________________________________________________________

[ x ]   help                    -   print help
[ x ]   cls                     -   clear screen
[ x ]   info                    -   print info about the system 
[   ]   date                    -   print date and time
[   ]   sleep '#'               -   sleep for # ms
[   ]   restart                 -   restart the system
[   ]   shutdown                -   shut down the system

[   ]   pwd                     -   print working directory
[   ]   ls                      -   list files
[   ]   cp 'src' 'dest'         -   copy file to destination
[   ]   mv 'src' 'dest'         -   move file to destination
[   ]   rm 'f1' 'f2' ...        -   remove file/folder
[   ]   md 'name'               -   make folder
[   ]   rd 'name'               -   remove folder
[   ]   cd 'path'               -   change directory
[   ]   find 'file'             -   find file/folder

[   ]   cat 'file' ...          -   print content of file
[   ]   echo 'string'           -   print string
[   ]   grep 'file' 'string'    -   find string in file
[   ]   dmp 'file'              -   binary dump of file
[   ]   less                    -   print less on screen

[   ]   sb 'offset' '#' ...     -   set bytes in memory 
[   ]   memdmp 'offset' 'rows'  -   dump memory
[   ]   memcpy  
[   ]   memset
[   ]   load 'offset' 'file'    -   load file at offset
[   ]   exec 'offset'           -   execute at offset
[   ]   debug '#'               -   set os debug mode
